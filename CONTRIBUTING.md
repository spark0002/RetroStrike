# 贡献指南 | Contributing Guide

## 中文

感谢你对 RetroStrike 倡议的关注！

### 如何参与

#### 传播倡议
- Star 本仓库
- 分享给你的朋友和社区
- 翻译倡议内容为更多语言

#### 监督记录
- 在 Issues 中提交你发现的企业作恶事件
- 提供可靠的信息来源和证据
- 使用标签 `monitor` 标记监督类 issue

#### 开源平替
- 在 Issues 中提议需要平替的商业软件
- 使用标签 `retrostrike` 标记平替项目提案
- 参与已有平替项目的开发

### 行为准则
- 基于事实，不传播未经验证的信息
- 尊重他人，理性讨论
- 独立开发，尊重知识产权
- 专注于建设，而非破坏

## English

Thank you for your interest in the RetroStrike initiative!

### How to Participate

#### Spread the Word
- Star this repository
- Share with your friends and communities
- Translate the initiative into more languages

#### Monitor & Document
- Submit corporate misconduct events via Issues
- Provide reliable sources and evidence
- Use the `monitor` label for oversight issues

#### Open-Source Alternatives
- Propose commercial software that needs open-source alternatives via Issues
- Use the `retrostrike` label for replacement project proposals
- Contribute to existing replacement projects

### Code of Conduct
- Stick to facts — do not spread unverified information
- Respect others — engage in rational discussion
- Develop independently — respect intellectual property
- Focus on building, not destroying
